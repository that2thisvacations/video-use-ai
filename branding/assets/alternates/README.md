# TravelBuddy Alternate Assets

This folder stores optional TravelBuddy branding assets that are not used by default in the demo workflow.

Use cases:

- backup watermark variants
- alternate footer or tag treatments
- experimental brand art that should stay out of the main asset path

Notes:

- these assets are optional
- they are not the canonical files the demo workflow expects
- if a future brand pack wants to use them, the wrapper or render layer can opt in explicitly
