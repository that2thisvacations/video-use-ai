# End Cards

Expected files:

- `travelbuddy_endcard.png`
- optional animated end card exports such as `travelbuddy_endcard.mp4` or `travelbuddy_endcard.webm`

Recommended format:

- transparent PNG for static cards
- MP4 or WebM for motion end cards
- use alpha-capable exports when the end card needs to float over video

Design notes:

- end cards should close the story cleanly
- keep them simple, readable, and brand-aligned
- make them easy to swap per theme or campaign
