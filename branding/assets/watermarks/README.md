# Watermarks

Expected files:

- `travelbuddy_watermark.png`
- optional alternate watermark variants for light and dark footage

Recommended format:

- transparent PNG with alpha
- tight crop around the mark
- high-contrast edges that survive scaling

Usage notes:

- watermark should sit in a corner, not over faces or titles
- keep it subtle enough to survive social exports
- if a future animated watermark is needed, use a separate MP4/WebM asset and keep the PNG as the fallback
